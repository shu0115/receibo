= ローカル環境動作手順

config/local_setting.rb作成

  LOCAL_OAUTH_CONSUMER_KEY = "YOUR_CONSUMER_KEY"
  LOCAL_OAUTH_CONSUMER_SECRET = "YOUR_CONSUMER_SECRET"

  rake db:migrate

  rails s
